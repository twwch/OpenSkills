# 年度复盘报告模板

请根据以下模板结构，将周报内容汇总生成年度复盘报告。

## 复盘说明

1. 坚持 "无记录，不复盘"：填写时需依托客观数据和实际记录，避免主观臆断；
2. 目标聚焦 SMART：回顾目标时需明确是否符合 "具体、可衡量、可达成、相关、有时限"，不模糊表述；
3. 闭环思维：行动计划需明确 "负责人、时间、验收标准"，并设置跟踪节点，确保落地；
4. 氛围原则：个人复盘聚焦 "学习与成长"，项目复盘聚焦 "问题与优化"，均需对事不对人，重视经验转化。

## 生成指南

### 从周报提取信息

1. **目标复盘**
   - 识别周报中提到的年度/季度目标
   - 统计目标相关工作的完成情况
   - 计算达成率（基于周报中的进度描述）

2. **工作亮点**
   - 提取周报中明确标注为"成果"、"完成"、"上线"的内容
   - 优先选择有数据量化的成果
   - 筛选真正有影响力的工作

3. **成就客户**
   - 识别周报中涉及客户、用户、业务方的工作
   - 提取为他人创造价值的案例

4. **协作贡献**
   - 提取培训新人、知识分享、文档沉淀相关内容
   - 识别流程优化、效率提升的工作

5. **成长进化**
   - 提取学习新技能、使用新工具的内容
   - 重点关注 AI 工具使用和创新实践
   - 关联这些创新带来的指标提升

6. **项目贡献**
   - 按项目维度汇总周报内容
   - 识别项目角色和贡献
   - 总结展现的能力项

7. **经验教训**
   - 提取周报中的复盘、反思内容
   - 总结成功经验和失败教训

### 写作风格

- 使用**客观、专业**的语言
- 数据要**准确、具体**，尽量量化
- 成果描述要**有说服力**，体现价值
- 经验教训要**有洞察**，可复用
- 遵循 **SMART 原则**

### 注意事项

- 确保信息准确，不要臆造不存在的内容
- 如果某些信息在周报中未提及，标注为"[待补充]"
- 敏感信息需要脱敏处理
- 突出重点，避免流水账
