# Word 文档处理指南

## docx 文件结构

docx 文件实际上是一个 ZIP 压缩包，包含：
- `word/document.xml` - 文档主要内容
- `word/styles.xml` - 样式定义
- `word/media/` - 图片等媒体文件

## 常见样式

| 样式名称 | 说明 |
|---------|------|
| Title | 文档标题 |
| Heading 1-9 | 标题级别 |
| Normal | 正文 |
| List Paragraph | 列表项 |

## 处理建议

1. **大文档**: 分段处理，避免内存溢出
2. **表格**: 注意合并单元格的处理
3. **图片**: 需要单独提取和处理
4. **批注**: 可以通过 comments.xml 获取

## 输出格式建议

- **总结**: 使用 markdown 格式，便于阅读
- **数据提取**: 使用 JSON 格式，便于程序处理
- **转换**: 保持原有结构，注意样式映射
