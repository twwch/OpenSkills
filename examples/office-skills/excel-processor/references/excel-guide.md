# Excel 数据处理指南

## xlsx 文件结构

xlsx 文件是 Office Open XML 格式的压缩包：
- `xl/workbook.xml` - 工作簿信息
- `xl/worksheets/` - 各工作表数据
- `xl/sharedStrings.xml` - 共享字符串
- `xl/styles.xml` - 样式定义

## 数据类型

| 类型 | 说明 | 处理方式 |
|------|------|----------|
| 数值 | 整数、浮点数 | 直接读取 |
| 文本 | 字符串 | 注意编码 |
| 日期 | Excel 日期序列号 | 需要转换 |
| 公式 | 计算公式 | 读取计算结果 |
| 布尔 | TRUE/FALSE | 转换为 bool |

## 常用统计分析

### 数值列
- **求和 (SUM)**: 所有数值相加
- **平均值 (AVG)**: 总和除以数量
- **最大值 (MAX)**: 最大的数值
- **最小值 (MIN)**: 最小的数值
- **计数 (COUNT)**: 非空单元格数量

### 文本列
- **唯一值数**: 去重后的数量
- **频率分布**: 各值出现次数
- **众数**: 出现最多的值

## 数据质量检查

1. **缺失值**: 检查空单元格
2. **异常值**: 检查超出正常范围的数据
3. **重复值**: 检查重复行
4. **格式一致性**: 检查同列数据格式

## 输出建议

- **小数据集** (< 100行): 可以完整展示
- **中等数据集** (100-1000行): 展示摘要和样本
- **大数据集** (> 1000行): 只展示统计信息和关键发现
