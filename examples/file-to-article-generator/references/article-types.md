## 类型概览

支持以下文章类型：

| 类型标识 | 类型名称 | 适用场景 | Prompt索引 |
|---------|---------|---------|-----------|
| `brand-event` | 品牌动向 | 品牌发布会、品牌合作、品牌升级、品牌活动 | brand-event-prompt |
| `product-feature` | 产品动态 | 新功能发布、产品更新、产品优化、技术改进 | product-feature-prompt |
| `product-policy` | 政策规则 | 平台规则更新、用户政策变化、服务条款调整 | product-policy-prompt |
| `product-campaign` | 营销战役 | 促销活动、营销战役、品牌推广、节日活动 | product-campaign-prompt |
| `success-case` | 客户证言 | 客户成功案例、客户评价、使用体验分享 | success-case-prompt |
| `industry-news` | 行业资讯 | 行业趋势、市场动态、竞品分析、行业报告 | industry-news-prompt |

## 类型详细说明

### 1. 品牌动向 (brand-event)
**特征**：
- 关注品牌层面的活动和变化
- 突出品牌价值和定位
- 影响范围较广，面向公众或合作伙伴
- 语言积极正面，塑造品牌形象

**适用场景**：
- 品牌发布会报道
- 品牌战略合作公告
- 品牌升级或重塑
- 品牌周年庆活动
- 品牌获奖或认证

### 2. 产品动态 (product-feature)
**特征**：
- 聚焦产品功能和特性
- 技术导向，强调创新和改进
- 面向用户和开发者
- 语言专业但易懂

**适用场景**：
- 新功能发布
- 产品版本更新
- 技术优化说明
- 性能改进说明
- 产品迭代总结

### 3. 政策规则 (product-policy)
**特征**：
- 说明政策或规则的变化
- 严谨正式，避免歧义
- 明确生效时间和适用范围
- 提供清晰的指导和说明

**适用场景**：
- 平台规则更新公告
- 用户政策调整说明
- 服务条款变更通知
- 隐私政策更新
- 使用规范发布

### 4. 营销战役 (product-campaign)
**特征**：
- 营销导向，吸引用户关注
- 突出优惠、促销或活动亮点
- 行动号召明确
- 语言有吸引力，激发参与意愿

**适用场景**：
- 促销活动宣传
- 节日营销战役
- 新品推广活动
- 会员专享活动
- 联合营销活动

### 5. 客户证言 (success-case)
**特征**：
- 以客户视角讲述故事
- 强调实际效果和价值
- 案例真实可感
- 语言亲切，易于共鸣

**适用场景**：
- 客户成功案例
- 客户使用体验分享
- 客户评价汇总
- 合作伙伴案例
- 行业标杆案例

### 6. 行业资讯 (industry-news)
**特征**：
- 关注行业整体动态
- 提供宏观视角和洞察
- 数据支撑，分析深入
- 语言客观专业

**适用场景**：
- 行业趋势分析
- 市场动态报道
- 竞品分析报告
- 行业研究报告
- 市场数据发布

## 判断标准

### 基于文件内容的判断

**特征关键词识别**：

| 类型 | 内容特征关键词 |
|-----|--------------|
| brand-event | 品牌、发布、合作、升级、周年、活动、战略、形象、定位 |
| product-feature | 功能、版本、更新、优化、特性、技术、性能、迭代、发布 |
| product-policy | 规则、政策、条款、规范、通知、公告、调整、生效、适用 |
| product-campaign | 促销、优惠、活动、战役、推广、节日、折扣、礼包、抢购 |
| success-case | 案例、客户、成功、体验、评价、见证、故事、合作、伙伴 |
| industry-news | 行业、趋势、市场、动态、报告、分析、数据、竞品、宏观 |

### 基于用户需求的判断

**需求关键词识别**：

| 类型 | 用户需求关键词 |
|-----|--------------|
| brand-event | 品牌发布、品牌活动、品牌新闻、品牌宣传、品牌合作 |
| product-feature | 新功能、产品更新、产品发布、功能介绍、技术说明 |
| product-policy | 规则更新、政策变化、条款调整、使用规范、通知公告 |
| product-campaign | 促销活动、营销活动、宣传推广、节日活动、优惠信息 |
| success-case | 客户案例、成功案例、客户评价、使用体验、案例分享 |
| industry-news | 行业资讯、市场动态、行业分析、行业报告、趋势分析 |

### 综合判断规则

1. **优先级**：用户需求 > 文件内容特征
   - 如果用户明确指定了文章类型，优先按照用户需求判断
   - 如果用户未明确指定，则根据文件内容特征判断

2. **特征匹配度**：
   - 统计文件内容中各类型的特征关键词数量
   - 选择关键词数量最多的类型
   - 如果多个类型关键词数量相近，结合用户需求进一步判断

3. **模糊处理**：
   - 如果特征不明确，建议用户从6种类型中选择
   - 提供每种类型的示例说明，帮助用户选择

### 判断示例

**示例1**：
- 文件内容：包含品牌发布会流程、品牌战略升级说明、嘉宾致辞
- 用户需求：生成一篇品牌活动报道
- 判断结果：`brand-event`（品牌动向）

**示例2**：
- 文件内容：产品新功能列表、技术优化说明、版本更新日志
- 用户需求：介绍产品新特性
- 判断结果：`product-feature`（产品动态）

**示例3**：
- 文件内容：平台使用规则更新、新政策生效时间、适用范围说明
- 用户需求：发布规则更新公告
- 判断结果：`product-policy`（政策规则）

**示例4**：
- 文件内容：促销活动时间、优惠力度、参与方式
- 用户需求：制作活动宣传文案
- 判断结果：`product-campaign`（营销战役）

**示例5**：
- 文件内容：客户使用产品后的效果、客户评价、合作历程
- 用户需求：展示客户成功案例
- 判断结果：`success-case`（客户证言）

**示例6**：
- 文件内容：行业趋势分析、市场数据、竞品对比
- 用户需求：发布行业分析报告
- 判断结果：`industry-news`（行业资讯）
